Deploy OAuth Worker

Deploy an OAuth proxy worker to enable Chrome extension authentication with this Shopify store.

Create at: dash.cloudflare.com → My Profile → API Tokens

Setup Instructions

  1. Get your Cloudflare API token with Workers edit permissions
  2. Find your Account ID in the Cloudflare dashboard
  3. Get your Chrome extension ID from chrome://extensions
  4. Click Deploy Worker
  5. Copy the worker URL to your extension configuration
